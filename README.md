OGUREZ – your ultimate PC remote control dashboard! 🚀

Here’s what it does:

🔹 100+ built‑in commands – system info, network tools, file ops, process/service control, volume, notifications, hashing, encoding, and more.
🔹 Beautiful glass‑morphism UI with dark theme, animated background, and neon OGUREZ logo.
🔹 Live visual stats – CPU, RAM, disk, battery with progress bars.
🔹 Task Manager – live process list with kill buttons, updates every 3s.
🔹 Screenshot capture – directly in the browser.
🔹 Clipboard reader, Wi‑Fi info, external IP, ping/tracert/nslookup.
🔹 Power controls – shutdown, reboot, sleep, hibernate, lock.
🔹 Sound controls – volume 0‑100%, mute, +/‑, beep.
🔹 File management – list, create, delete, copy, move, rename, zip/unzip.
🔹 Service management – start/stop/query Windows/Linux services.
🔹 Utilities – run apps (notepad, calc, cmd, etc.), eject drives, turn off monitor, send messages.
🔹 Hashing & encoding – MD5, SHA1, SHA256, Base64, URL encode/decode, random password generator.
🔹 Password‑protected – default 8471 (you can change it).
🔹 Accessible from any device on your local network (phone, tablet, laptop).
🔹 Splash screen with animated logo on load.
🔹 Fully in English (commands stay system‑agnostic).

To get started, you need:

🐍 Python 3.7+ installed.

📦 Flask – pip install flask

(Optional) For extra features:

psutil – for process list, disk, battery, CPU stats.

Pillow – for screenshots.

pycaw + comtypes – for volume control on Windows.

requests – for external IP and file downloads.

🔧 Download the script (app.py) and run:

bash
python app.py
🌐 Open in your browser: http://<your_local_ip>:8000

🔑 Enter password 8471 and enjoy!

All commands work right away – just click a button or type /command in the input field.

Happy controlling! 😎👾
